Common Pitfalls in Goal Setting

The way we set a goal can sometimes set us up for failure. There are three very common pitfalls people slip into when setting a goal: setting a compensatory goal, which is when we set a goal to "make up" for something in the past; setting a fantasy based goal, which is when we set a goal that is so far away that it is hard to experience positive reinforcement along the way; and setting a goal based on the wrong intentions, like living up to expectations of others. Understanding how we set goals in the wrong way can help us start setting inspiring and achievable goals.
